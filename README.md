# MAG Website

Mehrsprachige B2B-Website fur MAG (DE/EN/FR) auf Basis von Next.js App Router, TypeScript und Tailwind CSS v3.

## Start

```bash
npm install
npm run dev
```

## Routen

- `/` (DE), `/en`, `/fr`
- `/services/*`, `/en/services/*`, `/fr/services/*`
- `/company`, `/en/company`, `/fr/company`
- `/contact`, `/en/contact`, `/fr/contact`
- `/impressum`, `/datenschutz`
